# MAXX Workspace Router

## Purpose

Run Stacy's company operations through visible, reviewable ICM stages.

## Routing

1. Create a unique folder under `runs/`.
2. Begin with `01_intake`.
3. Advance one numbered stage at a time.
4. Write every intermediate result into the current stage's `output/`.
5. Stop at `06_approval` when the action has external consequences.
6. Record verification before delivery.
