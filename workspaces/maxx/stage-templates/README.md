# ICM Stage Templates

The control plane creates these contracts per run:

1. `01_intake`
2. `02_research`
3. `03_plan`
4. `04_execute`
5. `05_verify`
6. `06_approval`
7. `07_deliver`
8. `08_learn`

Each generated contract declares Inputs, Process, Outputs, Verify, Allowed Skills, Token Budget, and Approval Requirements.
