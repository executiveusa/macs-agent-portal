# Model Routing

Use the least costly healthy model that meets the mission's capability, context, tool, and reliability needs.

- Conversation and summaries: low-latency lane.
- Research: retrieval-capable lane.
- Planning: reasoning lane.
- Coding: tool-capable coding lane.
- Browser and vision: multimodal lane.
- Consequential actions: high-reliability lane plus approval.

Record the selected model, reason, token usage, latency, and fallback events.
