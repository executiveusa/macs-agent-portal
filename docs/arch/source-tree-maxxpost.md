# Source Tree & Tech Stack - MAXX-POST

## Tech Stack
- **Framework**: Vite + React + TypeScript
- **Styling**: Tailwind CSS, Shadcn UI
- **Animation**: Framer Motion (already in dependencies)
- **State/Data**: React Query, Supabase (client present)
- **Routing**: React Router DOM

## Directory Structure (Key Areas)
- `src/` - Application Source
  - `components/` - UI Components (Shadcn, Custom)
  - `pages/` - Route Pages
  - `hooks/` - Custom Hooks
- `public/` - Static Assets
  - `MUSTANG MAXX/` - Primary Asset Library
    - `006/` - Character Assets (Agent 006)
    - `COMIC/` - Narrative Panels / Backgrounds
    - `HERO FULL PAGE IMAGE/` - Main Hero
    - `MUSTANG MAXX/` - Vehicle Details
    - `THE MAXX/` - Branding/Logo

## Asset Mapping (007 -> MAXX)
- **Hero** -> `HERO FULL PAGE IMAGE/*.png`
- **Briefing** -> `006/*.png` (Agent 006 profile)
- **Car Features** -> `MUSTANG MAXX/*.png` (Car details)
- **Narrative/Transitions** -> `COMIC/*.png` (Noir scenes)
